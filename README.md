## Installation

Project Installation

```bash
  git clone https://github.com/Chim-dev/CareerPath.AI.git
```

Go to the project directory

```bash
  cd CareerPath
```

Install dependencies 

```bash
  npm install
```

Start the server

```bash
  npm run start
```
